# glasscordctl

glasscordctl is a terminal managment tool for glasscord

# usage

## Install Glasscord:

```bash
glasscordctl --install
```

## Update your installation:

```bash
glasscordctl --update
```

## Uninstall Glasscord:

```bash
glasscordctl --uninstall
```

## Reinstall Glasscord:

```bash
glasscordctl --reinsatll
```

## Print this message:

```bash
glasscordctl --help
```

## Print the version:

```bash
glasscordctl --version
```

## Print info:

```bash
glasscordctl --info
```
